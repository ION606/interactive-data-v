What is all this?

By moving the threshold slider, you decide how confident the model must be before flagging a transaction as fraud (pred 1). The precision-recall curve and key metrics update instantly to show the trade-off: a higher threshold reduces false positives (pred 1 on actual 0) but may increase false negatives (pred 0 on actual 1).
